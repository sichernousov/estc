Задача 6. Взаимодействие двух плат
Предлагается реализовать командный интерфейс между двумя платами STM32, используя UART в качестве транспорта.
Команды, которые можно выполнить:
1. Управление светодиодами – можно установить яркость светодиодов от 0 до 100%
2. Запустить таймер, по которому зарезервированный для этой цели светодиод начнет мигать
3. Изменить частоту таймера
4. Изменить время работы таймера в секундах. 0 – означает работать всегда
5. Запросить общий статус системы

Для связи двух плат надо замкнуть выходы (UART2):
PA2(1) - PA3(2)
PA3(1) - PA2(2)

Команды выполняющие необходимый функионал описаны в cmd.h
Обработчики кнопок в main.c
